# BDD & C++ Practice

Dieses Repository enthält meine Übungen zu C++, BDDs, ROBDDs und dem Apply-Algorithmus.  
Der Code umfasst:

- Grundlagen in C++ (Pointer, Klassen, Referenzen, Vektoren)
- Implementierung von einfachen BDD-Knoten
- Shannon Expansion Beispiele
- Kleine BDD-Traversierungsprogramme
- Mini-Implementierung von Apply + Unique Table (ROBDD Konzept)

Diese Sammlung wurde im Rahmen meiner Vorbereitung auf ein Gespräch an der Universität Bremen (AG Drechsler) erstellt.
